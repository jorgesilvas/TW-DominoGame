Intruções

  • Peças - 28 peças com lados pontuados de 0 a 6.
  • Distribuição - 7 peças para cada jogador. 14 peças no monte (para serem compradas).
  • Objetivo - Fazer 100 pontos.

Definições

Peça de dominó - É uma peça composta por duas pontas, cada uma com um número (exemplos de peças: 2-5, 6-6, 0-1).
Monte - Após a distribuição das peças para os jogadores, as 14 peças que sobram formam o monte, que é onde os jogadores poderão comprar durante a partida.
Encaixar peça - Quando uma peça é colocada ao lado de outra que tem pelo menos um número em comum (exemplo: 2-5 encaixa com 5-6).
Extremidades do jogo - São as peças livres das duas pontas do jogo, cujos lados estão em aberto para que outras peças sejam encaixadas.
Comprar peça - Quando o jogador não tem nenhuma peça que encaixe em qualquer extremidade.
Passar a vez - Quando o jogador não tem nenhuma peça para colocar numa das extremidades e o monte está vazio.
Jogo bloqueado - Quando nenhum jogador nao tem nenhuma peça que encaixe numa das extremidades.
Bloquear o jogo - Quando um jogador joga uma peça para bloquear o jogo.
Bater o jogo - Quando um dos jogadores consegue ficar sem peças na mão, tendo colocado todas as que possui.

     As peças são baralhadas na mesa, cada jogador pega 7 peças para jogar. O jogador que começa a partida é o que tem a dupla maior (6-6, 5-5, 4-4...), se nenhum jogador possuir nenhuma peça dupla, começa quem tiver a peça com valor mais alto (6-5, 6-4...).
    Inicia-se a partida colocando esta peça no centro da mesa. A partir daí, joga-se no sentido anti-horário. Cada jogador tenta encaixar uma peça sua numa das extremidade do jogo, uma em cada jogada.
    Caso o jogador não tenha nenhuma peça que encaixe numa das extremidades do jogo deve comprar peças do monte até que encontre uma peça que encaixe. Se o monte estiver vazio, passa a vez ao próximo jogador.
    A partida pode terminar em duas circustâncias:
      1- Quando um jogador consegue bater o jogo, ou quando o jogo fica bloqueado.
      2- Após a primeira partida, os jogadores alternam a vez de quem inicia cada jogo.

Contagem de Pontos

    Caso algum jogador tenha batido o jogo, obtem todos os pontos das peças que estão na mão do adversário. Caso o jogo fique bloqueado, conta-se todos os pontos conseguidos por cada jogador. O jogador que possuir menos pontos é o vencedor e leva todos os pontos do adversário.
    Caso haja um empate nesta contagem de pontos, o jogador que bloqueou o jogo perde e o vencedor leva todos os pontos deste jogador. Os pontos do vencedor são acumulados e o jogo termina quando alguém atinge a marca de 100 pontos.

Pontuação

    A pontuação de cada peça corresponde à soma dos valores das duas pontas da peça. Dessa forma, a peça 0-0 vale 0 pontos, a peça 3-4 vale 7 pontos, a peça 6-6 vale 12 pontos e assim por diante.
